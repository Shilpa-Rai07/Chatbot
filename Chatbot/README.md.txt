 Admission ChatBot

This is a simple terminal-based chatbot built using Python and NLTK. It helps users with information about MCA admissions like eligibility, fees, deadlines, hostel facilities, placements, and more.

Features

- Interactive command-line interface
- Intent-based responses using stemmed keyword matching
- Uses a JSON-based knowledge base
- Logs conversation history to a `.txt` file
- No internet or database required


Requirements

- Python 3.6+
- NLTK

Install NLTK (if not already):

"pip install nltk"


How to Run

1. Clone or download this repository.
2. Run the chatbot from your terminal:

"python chatbot.py"

3. Ask questions like:

   * "What is the eligibility for MCA?"
   * "How much are the fees?"
   * "Is hostel available?"
   * "Tell me about placements"

4. Type `quit` or `exit` to stop the chat.
